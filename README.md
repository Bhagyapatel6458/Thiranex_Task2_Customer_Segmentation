# Customer Segmentation Dashboard - Task 2

## 📊 Project Overview
Customer Segmentation project using Power BI
and Python K-Means Clustering as part of 
Data Analyst Internship at Thiranex.
The goal was to segment customers based on 
behavior and demographics and visualize 
key insights.

## 🛠️ Tools Used
- Power BI Desktop
- Python (Google Colab)
- Scikit-learn (K-Means Clustering)
- Pandas Library

## 📁 Dataset Information
- Source: Mall Customer Segmentation Dataset
- Records: 200 customers
- Fields: CustomerID, Gender, Age, 
  Annual Income, Spending Score

## 📈 Dashboard Features
- Gender Distribution (Pie Chart)
- Age Distribution (Bar Chart)
- Income vs Spending Score (Scatter Chart)
- Spending Score by Age (Column Chart)
- Annual Income by Gender (Donut Chart)
- Customer Segments K-Means Clustering 
  (Bubble Chart) 🎯

## 🤖 K-Means Clustering
- Used Python scikit-learn
- Created 5 customer segments
- Cluster 0 → Medium Income Medium Spenders
- Cluster 1 → High Income Low Spenders
- Cluster 2 → Low Income High Spenders
- Cluster 3 → High Income High Spenders
- Cluster 4 → Low Income Low Spenders

## 🔍 Key Insights
- 56% Male and 44% Female customers
- Most customers aged between 30-40 years
- 5 distinct customer segments identified
- High income doesn't always mean 
  high spending score
- Females have slightly higher 
  spending scores than males
